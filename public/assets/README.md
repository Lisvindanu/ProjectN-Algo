# Assets Directory

This directory contains static assets like images, icons, and other media files.

## Structure

- `icons/` - Application icons
- `images/` - Images used in the application
- `animations/` - Animation files (if any)

## Usage

Files in this directory are served as static assets and can be referenced in the application using absolute paths starting with `/assets/`.

Example:
```tsx
<img src="/assets/images/logo.png" alt="Logo" />
```
